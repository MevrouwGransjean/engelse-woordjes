# Woordjes leren

Interactief spel om Engelse en Nederlandse woordjes te oefenen.

## Speel het spel
https://mevrouwgransjean.github.io/engelse-woordjes/

## Gebruik
1. Kies een richting:
   - Engels → Nederlands  
   - Nederlands → Engels  
2. Raad het woord  
3. Je krijgt feedback per letter  
4. Probeer het woord binnen 6 pogingen te raden  

## Doel
Oefenen van woordenschat op een speelse manier.
